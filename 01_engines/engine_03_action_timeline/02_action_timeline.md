# Action Timeline

---

# The Action Timeline

Every action lives somewhere.

Our goal is not to memorize verb tenses.

Our goal is to understand the life of an action.

This Engine represents every action using a single visual model.

```
                    ACTION

──────────────────────────────────────────────────────────────►

PAST                 NOW                 FUTURE
```

Everything that happens to an action can be represented on this timeline.

---

# The Three Questions

Every action can be understood by answering three simple questions.

## 1. Where is the action?

```
Past

Now

Future
```

---

## 2. How does the action behave?

```
Point

Process

Habit
```

---

## 3. Is the action real?

```
Real

Possible
```

These three questions describe every action in English.

# Real Actions

A continuous line represents reality.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

Examples

```
I eat breakfast every day.

I am eating breakfast now.

I ate breakfast yesterday.

I will eat breakfast tomorrow.
```

These actions belong to the real timeline.

---

# Possible Actions

A segmented line represents imagination.

```
- - - - - - - - - - - - - - - - - - -
```

Examples

```
I would eat breakfast.

I could eat breakfast.

I might eat breakfast.
```

These actions do not describe reality.

They describe possibility.

---

# Conditional Actions

Sometimes one action depends on another.

```
Condition

──────────────►

Result
```

Example

```
If I buy bread today,

↓

I will eat toast tomorrow.
```

The first action creates the second one.

The timeline now contains two connected actions.

---

# The Action Never Changes

English does not constantly invent new actions.

The action remains the same.

Only its location changes.

```
WORK

↓

worked

↓

working

↓

will work

↓

would work
```

These are different views of the same action.

# Engine Principle

Do not ask:

"What tense is this?"

"Where does this action live?"
