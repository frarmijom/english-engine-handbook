# Conditional Actions

---

# Two Connected Actions

A conditional sentence does not describe one action.

It describes the relationship between two actions.

```
ACTION A

↓

ACTION B
```

The first action determines what happens to the second.

---

# The Conditional Model

Every conditional sentence answers one question.

```
What happens if this action occurs?
```

The relationship is always the same.

```
CONDITION

↓

RESULT
```

---

# Real Future

The condition is possible.

The result is expected.

```
ACTION A

━━━━━━━━━━►

ACTION B
```

Structure

```
IF

+

WHO + ACTION

↓

WHO + WILL + ACTION
```

Example

```
If I buy bread today,

I will eat toast tomorrow.
```

Meaning

The future depends on a real condition.

---

# General Truth

Whenever the condition happens,

the result also happens.

Structure

```
IF

+

WHO + ACTION

↓

WHO + ACTION
```

Examples

```
If you heat water,

it boils.
```

```
If the server loses power,

it shuts down.
```

Meaning

The relationship is always true.

---

# Imagined Present

Reality is different.

The speaker imagines another situation.

```
ACTION A

- - - - - -►

ACTION B
```

Structure

```
IF

+

WHO + ACTION (Past)

↓

WHO + WOULD + ACTION
```

Examples

```
If I had more time,

I would study English every day.
```

```
If I earned more money,

I would travel more.
```

Meaning

The condition is imaginary.

The result is imaginary.

---

# Imagined Past

The condition never happened.

The result never happened.

```
Past

- - - - - -►

Past Result
```

Structure

```
IF

+

WHO + HAD + ACTION

↓

WHO + WOULD HAVE + ACTION
```

Examples

```
If I had bought bread,

I would have eaten toast.
```

```
If the engineer had restarted the server,

the outage would have ended earlier.
```

Meaning

Both actions belong to an imagined past.

---

# Conditional Timeline

Real

```
ACTION

━━━━━━━━━━━━━━►

RESULT
```

Imagined

```
ACTION

- - - - - - -►

RESULT
```

The visual difference immediately tells us whether the situation belongs to reality.

---

# Reading a Conditional

When you find an IF,

do not read one sentence.

Read two actions.

```
Action A

↓

Action B
```

Ask yourself

```
Is Action A real?

↓

Does Action B depend on it?
```

# The Mental Model

Never memorize

```
First Conditional

Second Conditional

Third Conditional
```

Instead recognize

```
REAL CONDITION

↓

REAL RESULT
```

or

```
IMAGINED CONDITION

↓

IMAGINED RESULT
```

Everything else is simply the English structure that represents that relationship.

# Engine Reminder
