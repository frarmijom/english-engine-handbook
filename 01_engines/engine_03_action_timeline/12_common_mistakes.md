# Common Mistakes

---

# Most Mistakes Are Not Grammar Mistakes

Most English mistakes happen because the action was misunderstood.

The structure is usually a consequence.

Before correcting the sentence,

identify the action.

---

# Mistake 01 — Confusing a Habit with "Now"

❌

```
I am working every day.
```

✅

```
I work every day.
```

Ask yourself

```
Is it happening now?

or

Is it my normal routine?
```

---

# Mistake 02 — Confusing "Now" with a Habit

❌

```
I work now.
```

✅

```
I am working now.
```

The action is alive.

Use the Action Timeline.

---

# Mistake 03 — Using Yesterday with a Habit

❌

```
I work yesterday.
```

✅

```
I worked yesterday.
```

Yesterday belongs to history.

The action is finished.

---

# Mistake 04 — Forgetting the Future Marker

❌

```
I work tomorrow.
```

✅

```
I will work tomorrow.
```

Tomorrow belongs to the future.

---

# Mistake 05 — Confusing Reality with Possibility

❌

```
I will help if I had time.
```

✅

```
I would help if I had time.
```

The situation is imaginary.

Reality changed.

The action must move to the Possible Timeline.

---

# Mistake 06 — Confusing Possibility with Reality

❌

```
I could finished the report.
```

✅

```
I could finish the report.
```

Possible actions use the base action.

---

# Mistake 07 — Treating Every "If" as the Same

These two ideas are different.

```
If I buy bread,

I will eat toast.
```

Reality.

---

```
If I had bought bread,

I would have eaten toast.
```

Imagination.

Always identify the timeline first.

---

# Mistake 08 — Building the Wrong Relationship

Remember.

A conditional sentence always contains two actions.

```
Condition

↓

Result
```

Do not think about one sentence.

Think about two connected actions.

---

# Mistake 09 — Choosing Structure Before Meaning

Many students ask

```
Which tense should I use?
```

Ask instead

```
Where does the action live?
```

The structure comes afterwards.

---

# Mistake 10 — Memorizing Names

Many students remember

```
Present Perfect

Past Continuous

Future Perfect
```

But they cannot explain the action.

Instead remember

```
Finished

Still connected

Happening now

Habit

Planned

Possible
```

Meaning comes before terminology.

---

# Mistake 11 — Ignoring the Action Timeline

Whenever you are unsure,

return to the Timeline.

```
WHERE?

↓

HOW?

↓

REAL?
```

Those three questions solve most problems.

---

# Mistake 12 — Ignoring the Sentence Skeleton

The Action Timeline explains the action.

Engine 02 explains the sentence.

Always use both.

```
WHO

↓

ACTION

↓

TARGET

↓

DETAILS
```

The Action Timeline only changes the Action.

Everything else remains stable.

---

# Recovery Strategy

Whenever you get stuck,

follow these steps.

```
1.

Identify the action.

↓

2.

Locate it.

Past?

Now?

Future?

↓

3.

Is it real?

Or possible?

↓

4.

Choose the structure.

↓

5.

Build the sentence.
```

---

# Engine Reminder

Do not correct grammar.

Correct your understanding of the action.

When the action is understood,

the English structure usually becomes obvious.
