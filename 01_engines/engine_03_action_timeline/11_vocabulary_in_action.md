# Vocabulary in Action

---

# Learn Actions, Not Words

This Engine is about actions.

Every new verb should immediately travel through the Action Timeline.

Do not learn

```
work
```

Learn

```
I work.

I worked.

I am working.

I will work.

I would work.
```

The action becomes alive.

# One Action

Observe what changes.

```
WORK

↓

worked

↓

working

↓

will work

↓

would work
```

Now replace the action.

```
DEPLOY

↓

deployed

↓

deploying

↓

will deploy

↓

would deploy
```

The pattern never changes.

Only the action changes.

# Engine Reminder

Vocabulary is not a list of words.

Vocabulary is a collection of actions moving through time.

Learn one action.

Move it across the Action Timeline.

Repeat.
