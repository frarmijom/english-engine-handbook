# The Real Timeline

A real action belongs to reality.

It happened.

It is happening.

Or it will happen.

Real actions are represented with a continuous timeline.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

Everything on this timeline belongs to the real world.

---

# Finished Actions

The action happened.

It is complete.

```
●
```

Examples

```
I ate toast yesterday.

The deployment finished.

The engineer restarted the server.

The meeting ended at five.
```

---

# Actions in Progress

The action is alive.

```
──────────────
```

Examples

```
I am eating breakfast.

The deployment is running.

The team is investigating the incident.

The server is restarting.
```

---

# Habits

The action repeats naturally.

```
↺
```

Examples

```
I eat eggs almost every day.

She drinks coffee every morning.

The backup runs every night.

The monitoring system checks the servers continuously.
```

---

# Future Actions

The action has not happened yet.

It belongs to the real future.

```
NOW ─────────────► ●
```

Examples

```
I will eat toast tomorrow.

The deployment will start tonight.

We will restart the application.

The meeting will begin at ten.
```

---

# Future Processes

The action will already be happening.

```
NOW ─────────► ───────────────
```

Examples

```
I will be eating breakfast at eight.

The deployment will be running tonight.

The engineers will be working all weekend.
```

---

# Connected to the Present

The action happened before.

Its result is still visible.

```
●══════════════► NOW
```

Examples

```
I have finished breakfast.

The deployment has completed.

The customer has approved the change.

We have solved the issue.
```

---

# Continuing Until Now

The action started before.

It has never stopped.

```
════════════════════════════► NOW
```

Examples

```
I have worked here for ten years.

She has lived in Chile since 2020.

The application has been running all day.

The investigation has continued since Monday.
```

# Planned but Interrupted

Reality changed.

The action never happened.

```
●

✖
```

Examples

```
I was going to eat toast.

We were going to deploy tonight.

She was going to travel.

The engineer was going to restart the server.
```
