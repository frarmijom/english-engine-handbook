# Master Table 01 — The Action Timeline

| Where? | How? | Reality? | Meaning |
|----------|-------|----------|---------|
| Past | Point | Real | Finished action |
| Past | Process | Real | Was happening |
| Past → Now | Process | Real | Started before and continues |
| Past → Now | Point | Real | Finished but still relevant |
| Now | Habit | Real | Routine or fact |
| Now | Process | Real | Happening now |
| Future | Point | Real | Planned action |
| Future | Process | Real | Will be happening |
| Any Time | Any | Possible | Imagined or conditional action |

# Master Table 03 — Cognitive States

| Cognitive State | Question |
|-----------------|----------|
| Habit | Is this normal? |
| Happening Now | Is it happening now? |
| Finished | Is it over? |
| Still Relevant | Does it still matter? |
| Continues | Has it continued until now? |
| Planned | Will it happen? |
| Possible | Could it happen? |
| Conditional | Does another action control it? |

---

# Master Table 04 — Sentence Structures

| Cognitive State | Structure |
|-----------------|-----------|
| Habit | WHO + ACTION + DETAILS |
| Happening Now | WHO + BE + ACTION-ing + DETAILS |
| Finished | WHO + ACTION (Past) + DETAILS |
| Planned | WHO + WILL + ACTION + DETAILS |
| Possible | WHO + WOULD / COULD / MIGHT + ACTION + DETAILS |
| Connected Actions | IF + ACTION → RESULT |

# Master Table 09 — Action Flow

```
WHO

↓

ACTION

↓

Action Timeline

↓

TARGET

↓

DETAILS
```

The Action Timeline determines the behavior of the action.

The sentence skeleton remains unchanged.
