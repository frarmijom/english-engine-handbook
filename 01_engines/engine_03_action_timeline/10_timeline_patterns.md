# Timeline Patterns

---

# Patterns Build Fluency

Native speakers do not build every sentence from scratch.

They reuse patterns.

The Action Timeline is built from a small number of reusable patterns.

Learn the pattern.

Replace the vocabulary.

---

# Pattern 01 — Habit

```
WHO

+

ACTION

+

DETAILS
```

Examples

```
I work every day.

She studies English.

The server restarts automatically.

The backup runs every night.
```

---

# Pattern 02 — Action in Progress

```
WHO

+

BE

+

ACTION-ing

+

DETAILS
```

Examples

```
I am working.

She is cooking dinner.

The deployment is running.

The engineers are investigating the incident.
```

---

# Pattern 03 — Finished Action

```
WHO

+

ACTION (Past)

+

DETAILS
```

Examples

# Pattern 04 — Connected to Now

```
WHO

+

HAVE

+

ACTION

+

DETAILS
```

Examples

```
I have finished my work.

The customer has approved the request.

# Pattern 05 — Continuing Until Now

```
WHO

+

HAVE

+

BEEN

+

ACTION-ing

+

DETAILS
```

Examples

```

She has been studying since lunch.

The application has been running all day.

The engineers have been investigating the incident for hours.

ACTION

DETAILS

```
WHO

+

WILL

BE

ACTION-ing

+

DETAILS

Examples

# Pattern 08 — Planned but Interrupted

```
WHO

+

WAS / WERE GOING TO

+

ACTION

+

DETAILS
```

Examples

```

She was going to travel.

# Pattern 09 — Possible Action

```
WHO

+

WOULD / COULD / MIGHT / SHOULD

+

ACTION

+

DETAILS
```

Examples

```

She could travel.

We might deploy tomorrow.

You should restart the service.
```

---

# Pattern 10 — Real Condition

```
IF

+

ACTION

↓

WILL

+

ACTION
```

Examples

```
If I buy bread,

I will make breakfast.
```

```
If the deployment succeeds,

the application will become available.
```

---

# Pattern 11 — Imagined Situation

```
IF

+

ACTION

↓

WOULD

+

ACTION
```

Examples

```
If I had more time,

I would study every day.
```

```
If the server had more memory,

it would perform better.
```

---

# Pattern 12 — Imagined Past

```
IF

+

HAD

+

ACTION

↓

WOULD HAVE

+

ACTION
```

Examples

```
If I had bought bread,

I would have eaten toast.
```

```
If we had deployed yesterday,

the customer would have received the update.
```

---

# Pattern Recognition

When reading a sentence,

do not identify grammar.

Identify the pattern.

```
↓

Habit

↓

Process

↓

Finished

↓

Connected

↓

Future

↓

Possible

↓

Conditional
```

Patterns are much easier to remember than grammatical names.
