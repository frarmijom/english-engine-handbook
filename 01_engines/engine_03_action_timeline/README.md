# Engine 03 — Action Timeline

## Fundamental Question

> How does English locate an action in time?

---

## Purpose

This Engine introduces the **Action Timeline**, a cognitive model that explains how English represents actions across time.

Instead of memorizing grammatical names, you will learn to recognize:

- where an action lives
- how an action behaves
- whether the action is real or possible

The correct English structure becomes the consequence of understanding the action.

## Learning Objectives

After completing this Engine you will be able to:

- Understand the Action Timeline.
- Locate actions in the past, present and future.
- Distinguish real actions from possible actions.
- Build conditional actions naturally.
- Understand verb structures without relying on grammatical terminology.
- Apply the same cognitive model to any verb.
