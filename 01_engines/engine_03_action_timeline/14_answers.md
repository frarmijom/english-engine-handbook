# Final Challenge

There is no single correct solution.

The objective is to move one action naturally through the Action Timeline.

Evaluate every sentence using this checklist.

```
✓ Is the action correctly located?

✓ Is its behavior correct?

✓ Is it real or possible?

✓ Does the sentence follow the Engine 02 skeleton?

WHO

↓

ACTION

↓

TARGET

↓

DETAILS
```

If the answer is "Yes" to every question,

the sentence follows the English Engine model.
