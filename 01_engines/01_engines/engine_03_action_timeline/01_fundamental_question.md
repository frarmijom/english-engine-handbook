# The Fundamental Question

```
How does English locate an action in time?
```

This Engine answers that question.

Not by teaching verb tenses.

But by teaching how English naturally perceives an action.

---

# The Problem

Traditional English courses usually begin with names such as:

- Present Simple
- Present Continuous
- Present Perfect
- Past Perfect
- Future Continuous

Students often memorize these names without understanding the action they represent.

As a result, English becomes a collection of rules instead of a system of ideas.

---

# A Different Perspective

This Engine starts from a different question.

Instead of asking:

> "What tense is this?"

we ask:

> "What is happening to the action?"

The action may:

- happen now
- have already finished
- still be in progress
- happen regularly
- be planned
- be imagined
- depend on another action

These are natural human concepts.

Not grammatical terminology.

---

# The Action Timeline

Throughout this Engine, every verb will be represented using a single cognitive model.

```
The Action Timeline
```

Every action will be analyzed according to three simple questions.

```
WHERE?

Past

Now

Future
```

```
HOW?

Point

Process

Habit
```

```
REAL?

Real

Possible
```

By answering these three questions, the correct English structure becomes much easier to understand.

# The Goal

By the end of this Engine, you will no longer think in terms of:

- Present Simple
- Past Continuous
- Future Perfect

Instead, you will naturally recognize:

- where an action lives
- how an action behaves
- whether the action is real or hypothetical

The English structure will become a consequence of understanding the action.

Not something that must be memorized.

---

# Engine Law

> English does not organize verbs by names.

> English organizes actions by time, behavior, and reality.

Understanding the action is the first step toward choosing the correct structure.
