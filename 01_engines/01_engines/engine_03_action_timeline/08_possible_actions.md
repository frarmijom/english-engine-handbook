# The Possible Timeline

Not every action belongs to reality.

Sometimes we imagine.

Sometimes we predict.

Sometimes we advise.

Sometimes we wonder what could happen.

These actions belong to the Possible Timeline.

```
- - - - - - - - - - - - - - - - - - -
```

A segmented line represents actions that are not part of reality.

---

# Possible Actions

Possible actions answer questions such as

```
Could it happen?

Would it happen?

Might it happen?

Should it happen?

May it happen?
```

The action exists only as an idea.

---

# Could

Meaning

```
Possible

Capable

Available
```

Examples

```
I could help you.

We could deploy tomorrow.

The engineer could restart the server.

The customer could approve the request.
```

---

# Might

Meaning

```
Possible

Uncertain
```

Examples

```
It might rain.

The deployment might fail.

The application might restart automatically.

She might arrive later.
```

---

# May

Meaning

```
Possible

Allowed

Expected
```

Examples

```
You may enter.

The customer may approve the proposal.

The deployment may begin tonight.
```

---

# Should

Meaning

```
Recommended

Expected

Best option
```

Examples

```
You should investigate the logs.

We should restart the service.

The engineer should review the deployment.

I should study more English.
```

---

# Would

Meaning

```
Imagined

Expected

Conditional
```

Examples

```
I would help you.

She would travel more.

The deployment would succeed.

The server would restart automatically.
```

---

# Possible Future

Some actions belong to a future that is not guaranteed.

```
Today

- - - - - - - - -►

Possible Future
```

Examples

```
I might travel next year.

We could migrate to the cloud.

The customer may request another meeting.
```

---

# Imagined Present

The action is imagined now.

Examples

```
I would be working now.

She would be studying.

The deployment would be running.

The application would be available.
```

Reality is different.

The speaker imagines another reality.

---

# Imagined Past

The action never happened.

Examples

```
I would have helped.

She would have travelled.

The deployment would have succeeded.

The engineer would have restarted the server.
```

These actions belong entirely to imagination.

---

# Comparing Reality and Possibility

Reality

```
I deployed the application.
```

Possible

```
I could deploy the application.
```

---

Reality

```
The server restarted.
```

Possible

```
The server might restart.
```

---

Reality

```
We will investigate the issue.
```

Possible

```
We should investigate the issue.
```

---

# Reading Possible Actions

When you read a sentence, ask yourself

```
Is this action real?

↓

No.

↓

What kind of possibility is it?

Could?

Would?

Might?

Should?

May?
```

Once the possibility is identified,

the structure becomes natural.

# Engine Reminder

English does not separate reality from possibility by vocabulary alone.

It changes the structure of the action.

Always determine first

```
REAL

or

POSSIBLE
```

before building the sentence.
