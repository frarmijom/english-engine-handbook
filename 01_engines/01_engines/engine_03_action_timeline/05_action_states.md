# Action States

---

# The Life of an Action

Every action passes through different states.

The Action Timeline describes those states.

Traditional grammar gives them names.

This Engine explains their meaning.

---

# State 01 — Habit

The action belongs to your normal life.

```
↺
```

Questions

```
Is this something that normally happens?

Is it part of a routine?

Is it generally true?
```

Examples

```
I work every day.

She drinks coffee every morning.

The server restarts automatically.

Water boils at 100°C.
```

Meaning

The action is part of reality.

It is not happening right now.

It is simply true.

---

# State 02 — Happening Now

The action is alive.

```
──────────────
```

Questions

```
Is the action happening now?

Has it started?

Has it finished?
```

Examples

```
I am working.

She is studying.

The deployment is running.

The team is investigating the issue.
```

Meaning

The action is currently in progress.

---

# State 03 — Finished

The action belongs to history.

```
●
```

Questions

```
Did it happen?

Did it finish?

Is it over?
```

Examples

```
I worked yesterday.

The deployment finished.

The customer called.

The engineer restarted the server.
```

Meaning

The action happened.

It is complete.

---

# State 04 — Connected to Now

The action happened before.

Its effect still exists.

```
●════════════► NOW
```

Questions

```
Did it happen before now?

Does it still matter?

Is it still relevant?
```

Examples

```
I have finished the report.

She has already left.

We have deployed the application.

The customer has approved the change.
```

Meaning

The action belongs to the past.

Its consequence belongs to the present.

---

# State 05 — Started Before and Continues

The action began earlier.

It has never stopped.

```
════════════════════► NOW
```

Questions

```
Did it begin before now?

Is it still happening?
```

Examples

```
I have worked here for five years.

She has lived in Chile since 2020.

The application has been running all day.
```

Meaning

The action connects the past with the present.

---

# State 06 — Future

The action has not happened yet.

```
NOW ─────────────► ●
```

Questions

```
Will it happen?

Is it planned?

Is it expected?
```

Examples

```
I will work tomorrow.

The deployment will start tonight.

She will travel next week.
```

Meaning

The action belongs to the future.

---

# State 07 — Future in Progress

The future action is already imagined as a process.

```
NOW ─────► ───────────────
```

Questions

```
Will the action be in progress?

Will it already be happening?
```

Examples

```
I will be working at 8 AM.

The deployment will be running tonight.

They will be travelling tomorrow afternoon.
```

Meaning

The future is viewed from inside the action.

---

# State 08 — Planned but Interrupted

The action was expected.

Reality changed.

```
●

✖

(Not completed)
```

Examples

```
I was going to eat toast.

She was going to travel.

We were going to deploy tonight.
```

Meaning

The intention existed.

The action never happened.

---

# State 09 — Possible

The action exists only as an alternative.

```
- - - - - - - - - -
```

Questions

```
Could it happen?

Would it happen?

Might it happen?
```

Examples

```
I would help.

She could travel.

We might deploy tomorrow.

The engineer should investigate the issue.
```

Meaning

The action belongs to possibility.

Not to reality.

---

# State 10 — Conditional

One action depends on another.

```
ACTION A

↓

ACTION B
```

Examples

```
If I buy bread,

I will eat toast.
```

```
If I had bought bread,

I would have eaten toast.
```

Meaning

The first action controls the second.

---

# Engine Reminder

Every English verb represents one action.

Every action lives in one state.

The Action Timeline helps you recognize that state before choosing the English structure.
