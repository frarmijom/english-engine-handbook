# Action Properties

---

# Every Action Has Three Properties

Every action can be described using three simple properties.

```
WHERE?

↓

HOW?

↓

REAL?
```

Together, these three properties describe the complete behavior of an action.

---

# Property 01 — Where?

The first property answers one question.

```
Where does the action live?
```

Possible locations

```
PAST

NOW

FUTURE
```

Examples

```
I worked yesterday.

I work every day.

I will work tomorrow.
```

The action moves.

The timeline does not.

---

# Property 02 — How?

The second property describes the behavior of the action.

There are three fundamental behaviors.

---

## Point

A single event.

```
●
```

Examples

```
The deployment finished.

The server restarted.

The customer called.
```

---

## Process

The action occupies a period of time.

```
──────────────
```

Examples

```
I am working.

The application is starting.

The team is investigating the issue.
```

---

## Habit

The action repeats naturally.

```
↺
```

Examples

```
I work every day.

She drinks coffee every morning.

The backup runs every night.
```

---

# Property 03 — Reality

The third property answers another question.

```
Is this action real?
```

---

## Real

```
━━━━━━━━━━━━━━━━━━━━━━
```

Examples

```
I work here.

The server restarted.

We deployed yesterday.

The meeting starts at nine.
```

---

## Possible

```
- - - - - - - - - - -
```

Examples

```
I would work.

I could help.

She might travel.

We should investigate.
```

Possible actions do not describe reality.

They describe possibility.

Whenever you see a verb, ask yourself:

```
WHERE?

↓

HOW?

↓

REAL?
```

Once these three questions are answered,

the correct English structure becomes much easier to understand.
