
This mental model is the foundation of clear communication.

---

# The Four Big Ideas

## 1. English Builds Ideas

Words alone do not communicate.

Ideas do.

Every sentence begins as a simple idea.

---

## 2. Start Small

Do not try to build a long sentence immediately.

Begin with the smallest complete idea.

```
The server restarted.
```

Then expand it naturally.

```
The server restarted yesterday.

↓

The server restarted yesterday after the deployment.

↓

The server restarted yesterday after the deployment because of a configuration error.
```

---

## 3. Learn Patterns, Not Rules

Fluent speakers reuse patterns.

They do not invent every sentence from scratch.

Examples

```
WHO + ACTION

WHO + ACTION + TARGET

WHO + BE + DESCRIPTION

THERE IS / THERE ARE

WHO + ACTION + WHAT + TO WHOM
```

Patterns reduce complexity.

---

## 4. Learn Vocabulary in Context

Words are never isolated.

They naturally connect with other words.

Instead of learning:

```
deploy
```

Learn:

```
deploy an application

deploy to production

deploy automatically

successful deployment
```

Natural English is built from natural combinations.

---

# Your New Thinking Process

When speaking English, ask yourself:

```
Who?

↓

What happened?

↓

To whom or to what?

↓

Anything else?
```

If you can answer these questions, you can usually build the sentence.

---

# Key Takeaways

✓ English organizes ideas through word order.

✓ Every sentence grows progressively.

✓ Simple ideas are easier to expand than complex ones.

✓ Vocabulary is learned through usage, not translation.

✓ Patterns are more valuable than isolated grammar rules.
