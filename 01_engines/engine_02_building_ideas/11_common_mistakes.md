# Common Mistakes

---

# Thinking in Spanish

The most common mistakes are not grammatical.

They are conceptual.

English and Spanish organize ideas differently.

Learning to recognize these differences is one of the fastest ways to improve.

---

# Mistake 1 — Translating Word by Word

❌

```
I have 43 years.
```

✅

```
I am 43 years old.
```

English does not always express ideas the same way as Spanish.

Translate ideas.

Not words.

---

# Mistake 2 — Building Long Sentences Too Early

❌

```
Yesterday because the deployment finished late the engineer decided to restart the server after talking with the customer.
```

✅

```
The deployment finished late.

The engineer talked to the customer.

Then he restarted the server.
```

Build the idea first.

Expand it later.

---

# Mistake 3 — Forgetting the Actor

❌

```
Restarted the server.
```

✅

```
The engineer restarted the server.
```

Every action needs a clear actor.

Ask yourself:

```
Who?
```

---

# Mistake 4 — Missing the Target

❌

```
The engineer fixed.
```

✅

```
The engineer fixed the server.
```

Many actions naturally affect something.

Complete the idea.

---

# Mistake 5 — Adding Too Many Details

❌

```
The engineer yesterday after lunch in production manually carefully restarted the server.
```

✅

```
The engineer restarted the server manually yesterday.
```

Details should help communication.

Not make it difficult.

---

# Mistake 6 — Memorizing Rules Instead of Patterns

❌

```
Subject

Verb

Object

Rule...

Rule...

Rule...
```

✅

Think

```
Who?

↓

What happened?

↓

To whom?

↓

Anything else?
```

Patterns are easier to remember than rules.

---

# Mistake 7 — Studying Isolated Words

❌

```
deploy

restart

monitor
```

✅

```
deploy an application

restart the server

monitor the system
```

Words naturally belong together.

Learn complete expressions.

---

# Mistake 8 — Ignoring Context

Many words change naturally depending on the situation.

Instead of learning:

```
issue
```

Learn

```
investigate an issue

report an issue

resolve an issue

identify an issue
```

Context gives vocabulary meaning.

---

# Mistake 9 — Trying to Sound Advanced

Simple ideas are easier to understand.

Native speakers constantly use simple structures.

Clarity is more important than complexity.

---

# Mistake 10 — Forgetting the Mental Model

Whenever you feel lost, return to the Engine.

```
WHO

↓

ACTION

↓

TARGET

↓

DETAILS
```

If those four elements are clear, the sentence will usually build itself naturally.

---

# Engine Reminder

Do not fight the language.

Follow its way of organizing ideas.

English becomes much easier when you think before you translate.
