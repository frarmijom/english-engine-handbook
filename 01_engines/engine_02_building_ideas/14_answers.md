# Final Challenge

There is no single correct answer.

Evaluate your sentences using this checklist.

```
✓ Is the actor clear?

✓ Is the action clear?

✓ Is the target clear?

✓ Are the details useful?

✓ Does the sentence express one complete idea?
```

If you answered "Yes" to all five questions, your sentence follows the mental model introduced in this Engine.
