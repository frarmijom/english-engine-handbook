# Mental Model

---

# English builds ideas progressively.

When people begin learning English, they often try to build the entire sentence at once.

Native speakers rarely do that.

Instead, an idea grows one piece at a time.

Imagine building with LEGO blocks.

You do not start with the roof.

You begin with a solid foundation.

English works in exactly the same way.

---

# Every idea begins with someone or something.

```
WHO?
```

Examples

```
The engineer

The server

The customer

My friend

The deployment
```

At this point, nothing has happened yet.

You only know who or what the idea is about.

---

# Then something happens.

```
WHO

↓

ACTION
```

Examples

```
The server restarted.

The customer called.

The engineer arrived.

The deployment finished.
```

Now a complete idea already exists.

It is small, but complete.

---

# Sometimes the action affects something else.

```
WHO

↓

ACTION

↓

TARGET
```

Examples

```
The engineer restarted the server.

The customer opened a ticket.

The deployment created an incident.

The application stored the data.
```

The idea becomes richer.

But it still follows the same structure.

---

# Finally, we can add details.

```
WHO

↓

ACTION

↓

TARGET

↓

DETAILS
```

Examples

```
The engineer restarted the server yesterday.

The application stored the data automatically.

The deployment finished in five minutes.

The customer opened a ticket this morning.
```

Notice something important.

The sentence was already complete before the details.

Details make the idea richer.

They do not create the idea.

---

# The Building Rule

Never try to build a long sentence immediately.

Instead, build it one layer at a time.

```
The server.

↓

The server restarted.

↓

The server restarted yesterday.

↓

The server restarted yesterday after the deployment.

↓

The server restarted yesterday after the deployment because of a configuration error.
```

Each new piece expands an idea that already exists.

---

# Think Like a Native Speaker

Do not ask yourself:

> "Where do I put the verb?"

Ask yourself:

> Who?

↓

> What happened?

↓

> To whom?

↓

> Anything else?

If you answer those four questions naturally, English will usually organize itself correctly.

That is the mental model behind this Engine.
