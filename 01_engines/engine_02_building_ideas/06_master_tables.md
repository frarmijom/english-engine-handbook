# Master Tables

---

# Master Table 01 — Core Sentence Patterns

| Purpose | Pattern | Example |
|----------|----------|---------|
| Describe something | WHO + BE + DESCRIPTION | The server is online. |
| Express an action | WHO + VERB | The deployment finished. |
| Affect something | WHO + VERB + TARGET | The engineer restarted the server. |
| Express existence | THERE IS / THERE ARE | There are three servers. |
| Give something | WHO + VERB + WHAT + TO WHOM | She sent the report to the customer. |

---

# Master Table 02 — Building an Idea

| Step | Question | Example |
|------|----------|---------|
| 1 | Who? | The engineer |
| 2 | What happened? | restarted |
| 3 | To what? | the server |
| 4 | When? | yesterday |
| 5 | Where? | in production |
| 6 | Why? | after the deployment |
| 7 | How? | manually |

---

# Master Table 03 — Idea Expansion

| Level | Sentence |
|---------|----------|
| 1 | The server restarted. |
| 2 | The server restarted yesterday. |
| 3 | The server restarted yesterday after the deployment. |
| 4 | The server restarted yesterday after the deployment because of a configuration error. |
| 5 | The server restarted yesterday after the deployment because of a configuration error during maintenance. |

---

# Master Table 04 — Common Details

| Question | Examples |
|----------|----------|
| When? | yesterday, today, this morning, last night, after lunch, two hours ago |
| Where? | in production, at home, at work, in the office, on the server |
| Why? | because of an error, because of maintenance, due to a failure |
| How? | automatically, manually, successfully, carefully, remotely |
| With whom? | with the team, with the customer, with my manager |

# Master Table 05 — Native Sentence Growth

| Start Here | Then Add | Finally Expand |
|-------------|----------|----------------|
| WHO | ACTION | TARGET |
| WHO + ACTION | DETAILS | MORE DETAILS |
| Small idea | Complete idea | Rich communication |

---

# Master Table 06 — Question Flow

When building any sentence, always ask yourself:

```
Who?

↓

What happened?

↓

To whom or to what?

↓

When?

↓

Where?

↓

Why?

↓

How?
```

Do not try to answer every question.

Answer only the ones your idea actually needs.

---

# Engine Reminder

English does not require long sentences.

It requires clear ideas.

Start with the smallest complete idea.

Then expand it naturally.
