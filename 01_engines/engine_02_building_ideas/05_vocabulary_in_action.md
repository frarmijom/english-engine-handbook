# Vocabulary in Action

---

# Words Live Together

One of the biggest mistakes English learners make is studying words in isolation.

For example,

```
deploy = desplegar
```

This is not enough.

Words naturally connect with other words.

Learning those connections is what makes your English sound natural.

This section teaches vocabulary the way native speakers use it.

---

```

```

## Common Destinations

```

```

## Common Modifiers

```

deploy remotely
```

## Typical Sentences

```

The deployment failed after five minutes.

The application was deployed to production.
```

# A New Way to Learn Vocabulary

Do not memorize isolated words.

Learn complete building blocks.

Instead of remembering:

```
deploy
```

Remember:

```
deploy an application

deploy to production

deploy automatically

deployment failed

successful deployment
```

This is how vocabulary becomes natural.
