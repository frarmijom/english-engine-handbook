# ENGINE 02 — Building Ideas

---

# Fundamental Question

How does English build ideas?

Knowing words is not enough.

Communication happens when words are organized into meaningful ideas.

This Engine explains how English transforms individual words into complete thoughts.

It is not about memorizing grammar.

It is about understanding how English naturally builds meaning.

---

# Why This Matters

Imagine that you know these words.

```
engineer

server

restart
```

Do they communicate anything?

No.

Now look at this.

```
The engineer restarted the server.
```

The words did not change.

Only their organization changed.

Meaning appeared.

English is not simply a collection of words.

English is a system for organizing ideas.

---

# Cognitive Objective

After completing this Engine, your first thought should no longer be:

```
Subject

Verb

Object
```

Instead, your brain should naturally ask:

```
WHO?

↓

What happened?

↓

To whom or to what?

↓

Anything else?
```

This is how ideas begin to take shape.

Grammar names can come later.

---

# Mental Model

Every complete idea in English grows progressively.

```
WHO

↓

ACTION

↓

TARGET

↓

DETAILS
```

The first three elements create the core idea.

Everything else expands it.

---

# Native Thinking

Native speakers rarely think about grammatical labels.

Instead, they naturally organize information.

```
Someone

↓

does something

↓

to someone or something

↓

somewhere

↓

sometime

↓

somehow
```

English uses word order to make these relationships clear.

---

# Main Concept

English builds meaning one layer at a time.

Start with the smallest complete idea.

Then expand it.

Never try to build a long sentence immediately.

Think of sentence construction as adding information to an already complete idea.
