# English Engine Handbook

A visual engineering handbook for understanding how English works.

This project is not a traditional grammar book.  
Its goal is to model English as a set of conceptual engines that explain how the language works from the inside.

## Core idea

English should not be learned only as a list of rules.

It should be understood as a system.

Each engine answers one fundamental question about the language.

## Learning philosophy

Understand first.  
Memorize later.  
Use naturally.
