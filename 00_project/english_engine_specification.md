# English Engine Specification

## Purpose

English Engine is a visual handbook that explains English through conceptual engines.

It does not organize English by grammar chapters.

It organizes English by mental operations.

## Engine definition

An engine is a visual module that answers one fundamental question about English.

Each engine must include:

1. Main question
2. Mental model
3. Native thinking
4. Visual diagram
5. Analogies
6. Real examples
7. Common mistakes
8. Engine law
9. Related engines

## Engine quality standard

An engine is ready when:

- It explains the why before the rule.
- It can be understood visually.
- It uses real examples.
- It avoids unnecessary theory.
- It helps the learner think in English.
- It connects to other engines.

## Design principle

Every square centimeter must teach something.

Whitespace is allowed only when it improves reading.
