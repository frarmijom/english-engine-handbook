# English Engine Knowledge Architecture

English Engine is organized as a network of conceptual engines.

The project does not teach grammar as isolated rules.

The project models how English works.

Every engine belongs to a knowledge domain.

---

## Level 1 — Foundation

These engines explain how English sees reality.

| Engine | Name | Fundamental question |
|---|---|---|
| 01 | Classification Engine | How does English classify things? |
| 02 | Sentence Engine | How does English build an idea? |
| 03 | Time Engine | How does English perceive time? |

---

## Level 2 — Representation

These engines explain how English represents reality.

| Engine | Name | Fundamental question |
|---|---|---|
| 04 | Action Engine | How does English represent actions? |
| 05 | Movement Engine | How does English represent movement? |
| 06 | Space Engine | How does English represent location and space? |
| 07 | Quantity Engine | How does English express amount and number? |
| 08 | Description Engine | How does English describe people, things, and situations? |

---

## Level 3 — Communication

These engines explain how English exchanges information.

| Engine | Name | Fundamental question |
|---|---|---|
| 09 | Question Engine | How does English request information? |
| 10 | Negation Engine | How does English deny or block an idea? |
| 11 | Modal Engine | How does English express possibility, ability, permission, and obligation? |
| 12 | Comparison Engine | How does English compare ideas? |
| 13 | Communication Engine | How does English report, explain, and exchange messages? |
| 14 | Perception Engine | How does English express noticing, realizing, seeing, and understanding? |
| 15 | Connection Engine | How does English connect ideas? |
| 16 | Condition Engine | How does English build hypothetical situations? |

---

## Level 4 — Advanced Time and Natural English

These engines help the learner move from functional English to natural English.

| Engine | Name | Fundamental question |
|---|---|---|
| 17 | Present Perfect Engine | How does English connect past and present? |
| 18 | Spanish Speaker Error Engine | Where does Spanish interfere with English? |
| 19 | Phrasal Verb Engine | How does English combine verbs and particles? |
| 20 | Collocation Engine | Which words naturally live together? |
| 21 | Conversation Engine | How does English sound natural in real interaction? |
| 22 | Thinking in English Engine | How does the learner stop translating? |
