# Design System

## Format

- Target format: A4 printable poster
- Secondary format: 16:9 digital poster
- Style: technical infographic
- Density: high, but readable

## Visual philosophy

The page should look like an engineering panel, not a textbook page.

## Color semantics

- Blue: structure / system
- Green: correct examples
- Red: mistakes / warnings
- Yellow: key insights
- Gray: notes / dependencies

## Layout principles

- Use compact sections.
- Prefer diagrams over paragraphs.
- Avoid decorative images.
- Use consistent positions across engines.
- Keep the engine law visible at the bottom.

## Reading levels

Each engine must support:

1. 5-second scan
2. 30-second review
3. 5-minute study
4. Deep analysis
